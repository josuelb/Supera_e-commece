# Supera_e-commece
 Backend de um site E-commerce, com utilização de __Django__.
 
# Requisitos mínimos como iniciar
  . Ter o Python3 instalado na maquina. <br>
  . Execultar o arquivo **.sh**, para iniciar.<br>
  
 # Ferramentas usadas
  . Django;<br>
  . Rest Framework; <br>
  . SimpleJWT[crypto]; <br>
  . PostgreSQL.<br>
